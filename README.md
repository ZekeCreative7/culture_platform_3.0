# Lina Life Insurance 조직문화 진단 및 분석 플랫폼 (Lina Culture Platform 3.0)

이 프로젝트는 Lina Life Insurance의 조직문화 개선 세션(팀빌딩, 팀장 세션, 크로스펑셔널 세션)을 효율적으로 운영하고 진단하기 위한 **단일 운영자용 브라우저 기반 분석 콘솔**입니다.

## 📌 주요 기능
1. **세션 및 일정 관리 (Sessions)**: 기수별 세션(8주 팀빌딩 / 4주 팀장 / 6주 크로스펑셔널)을 개설하고 조직도를 기반으로 참여 대상을 지정 및 관리합니다.
2. **설문 설계 및 배포 (Survey Creator)**: 구글 폼 기반 혹은 자체 문항(5점 척도/주관식)을 설계하고, 참여 링크 복사 및 QR 코드를 즉석에서 생성합니다.
3. **CSV 데이터 업로드 (CSV Upload)**: 개인 식별 정보(이름, 이메일, 사번 등)를 자동으로 제거하여 익명성을 보장한 채 설문 응답 CSV 데이터를 검증 및 업로드합니다.
4. **문항별 응답 분석 (Analytics)**: 문항별 평균 점수, 통계 정보, 정량 응답 분포를 조회합니다.
5. **변화 분석 리포트 (Report)**: 
   - **현 상황 진단**: 4대 영역(심리적 안전감, 사일로 해소, 회복탄력성, 전반 분위기)의 점수를 시각적으로 제공.
   - **퍼실리테이션 가이드**: 점수 구간에 맞춘 정교한 가이드라인 매칭 제공.
   - **변화 분석**: 사전/중간/사후 시점 간의 변화 양상을 Slope Chart 및 Dumbbell Chart로 한눈에 추적. (N<3 조건 시 엄격한 익명 마스킹 적용)
   - **정성 응답 AI 분석**: 주관식 답변에서 키워드 필터링 및 톤앤매너 요약을 추출하여 카드로 렌더링.
   - **출력 기능**: Excel 데이터 내보내기 및 A4 규격 최적화 PDF 리포트 인쇄 지원.

## 📂 폴더 구조

```text
├── webapp/                 # 메인 웹 어플리케이션 (Frontend SPA)
│   ├── index.html          # 서비스 메인 진입점
│   ├── src/
│   │   ├── app.js          # 전역 라우터 및 레이아웃 오케스트레이터
│   │   ├── state.js        # 전역 상태 관리 및 Firestore 동기화 엔진
│   │   ├── views/          # 캡슐화된 뷰 컴포넌트 (Deep Modules)
│   │   │   ├── sessions.js # 세션 관리 화면
│   │   │   ├── org.js      # 조직 관리 화면
│   │   │   ├── survey.js   # 설문 설계 및 배포 화면
│   │   │   ├── upload.js   # CSV 업로드 검증 화면
│   │   │   ├── analytics.js# 문항 분석 화면
│   │   │   └── report.js   # 변화 분석 리포트 화면
│   │   └── styles.css      # 디자인 시스템 기반의 메인 CSS 스타일시트
├── backups/                # 로컬 데이터 백업 파일 보관함
├── legacy_prototype/       # 과거 Streamlit 프로토타입 백레벨 참조 소스
└── README.md               # 플랫폼 안내서 (본 파일)
```

## 🚀 로컬 실행 방법
이 프로젝트는 빌드 시스템이 필요 없는 순수 바닐라 JS(ES Modules) SPA로 구성되어 있습니다. 로컬 캐시 꼬임 방지 등을 위해 로컬 웹 서버 구동을 권장합니다.

1. **로컬 실행**:
   ```bash
   cd webapp
   python3 -m http.server 4173
   ```
2. **브라우저 접속**:
   [http://localhost:4173](http://localhost:4173) 주소로 이동합니다.
