# 세션 리뷰 템플릿

이 문서는 **채팅창 1개당 1부**씩 만든다.

사용법:

1. 아래 프롬프트를 해당 채팅창에 붙여넣는다.
2. AI가 출력한 결과를 복사한다.
3. `feedback/sessions/NN-주제.md` 파일로 저장한다.

```text

목적은 칭찬이나 비판이 아니라, 다음 프로젝트에서 더 좋은 작업 프로세스와 에이전트 구성을 설계하는 거야.

- 이 창에서 무엇을 만들었는지
```
