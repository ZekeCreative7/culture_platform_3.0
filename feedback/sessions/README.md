# 세션 기록 저장 규칙

이 폴더에는 각 채팅창에서 뽑아낸 회고 기록을 저장한다.

## 파일명 규칙

`NN-짧은-주제.md`

예:

- `01-initial-prototype.md`
- `02-firebase-auth-and-sync.md`
- `03-analysis-report-pdf.md`
- `04-mobile-login-and-responsive-fixes.md`

번호는 가능한 한 실제 작업 순서에 맞춘다. 정확한 순서를 모르면 대략적인 순서로 두고, 파일 안에 “순서 불확실”이라고 적는다.

## 한 파일에 반드시 들어갈 것

- 이 창에서 하려던 목표
- 실제로 끝난 결과
- 바뀐 기능/파일
- 잘한 점
- 잘못한 점 또는 마찰
- 재작업/되돌림
- 검증 방식
- 다음에 반복하지 않을 교훈
- 이 세션에서 적합했던/부족했던 에이전트 역할

## 저장 전 체크

- [ ] 사실과 해석을 나눴다.
- [ ] 커밋/푸시 여부를 적었다.
- [ ] 검증 여부를 적었다.
- [ ] 재작업이 있었다면 원인을 적었다.
- [ ] 다음에 쓸 수 있는 교훈을 한 줄 이상 적었다.
